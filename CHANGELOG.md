# Changelog

All notable changes to BrowniGAT will be documented in this file.

## [0.5.0] - 2026-06-04

### Added

- Resource-aware scheduler controls with priority hints, CPU and GPU slot requests, and promotion-gate semantics.
- Synthetic biology design workflow with gene program design, pathway rewiring summaries, and construct blueprints.
- Stronger public OSS-facing metadata including application framing, demo workflows, issue templates, PR template, CI workflow, roadmap, and citation metadata.

## [0.4.0] - 2026-06-04

### Added

- Resource-aware scheduler metadata including priority, CPU and GPU slot requests, and promotion-gate support.
- Synthetic biology design layer with gene program design, pathway rewiring plans, and construct blueprints.
- Public project-maintenance files including issue templates, PR template, contributing guide, roadmap, citation metadata, and CI workflow.

## [0.3.0] - 2026-06-04

### Added

- Experiment scheduler control plane with stage planning, run queues, failure recovery, and scheduler artifacts.
- Foundation engine data plane with manifest partitioning, data sharding, worker-aware sampling, and stage-wise checkpoint retention.

## [0.2.0] - 2026-06-03

### Added

- Foundation-model workspace builder, multimodal trainer skeleton, checkpoint schema, validation lifecycle, and artifact indexing.
- Research-infra versus engine split for pretraining and runtime orchestration.

## [0.1.0] - 2026-06-03

### Added

- BrowniGAT baseline target-prioritization pipeline with GAT, GCN, GraphSAGE, and centrality baselines.
- Benchmark plotting, toy datasets, and first-pass vNext multimodal target discovery workflow.
